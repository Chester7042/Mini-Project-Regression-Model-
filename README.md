Mini project ini bertujuan membangun model prediksi harga rumah (*SalePrice*) menggunakan pendekatan *regression modeling*. Dataset awal terdiri dari 1.459 baris dan 81 kolom, kemudian dilakukan *feature selection* untuk menyederhanakan model dan mengurangi kompleksitas dengan memilih 9 variabel yang paling relevan, yaitu **LotArea, YearBuilt, OverallQual, OverallCond, GrLivArea, MiscFeature, GarageArea, KitchenQual**, dan **SalePrice** sebagai target. Setelah tahap *preprocessing*, beberapa model regresi diuji, yaitu **Linear Regression**, **Lasso Regression**, **Support Vector Regression (SVR)**, dan **XGBoost Regressor** untuk membandingkan performa prediksi. Evaluasi model menggunakan metrik **R² (R-squared)** karena metrik ini menunjukkan seberapa besar proporsi variasi harga rumah yang dapat dijelaskan oleh fitur-fitur dalam model, sehingga cocok untuk menilai kualitas model regresi secara keseluruhan dan memudahkan perbandingan antar model. Hasil eksperimen menunjukkan bahwa **XGBoost** memperoleh nilai R² tertinggi, yang mengindikasikan kemampuannya menangkap hubungan non-linear dan interaksi antar variabel lebih baik dibandingkan model regresi linear, sehingga memberikan performa prediksi yang paling optimal pada mini project ini.
